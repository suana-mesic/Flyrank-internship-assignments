### 1. Hero
- **Sections in order:** the one-line claim as the headline; a one-sentence sub-line naming the stack (C#, ASP.NET Core, SQL); primary button.
- **Call to action:** **"View my code on GitHub →"** (the one action, front and center).

### 2. Work (strongest case leads)
- **Sections in order:**
  1. **BookVerse** — my strongest project. E-commerce backend in C# with Clean Architecture, a decoupled notification background service, and an order-lifecycle state machine. A real code screenshot + the calm architecture diagram.
  2. **RFID attendance system** — secondary project, shown briefly.
- **Call to action:** **"See the full BookVerse repo →"** (links to GitHub).

### 3. About
- **Sections in order:** a short, human paragraph (who I am, third-year CS student, what I care about — databases and backend); a real photo; my skills grouped (Programming, Databases, Web, Mobile).
- **Call to action:** **"Read the code behind the claims →"** (GitHub).

### 4. Contact
- **Sections in order:** one line inviting contact; email; GitHub; LinkedIn; CodeWars.
- **Call to action:** **"Let's talk — or start with my GitHub →"**.

Every page sends the visitor to the same place: the code.
