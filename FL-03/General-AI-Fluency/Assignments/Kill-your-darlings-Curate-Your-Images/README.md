## Rejection note (the graded part)

**I rejected an AI-generated architecture diagram.** An earlier version of the Clean Architecture visual came in loud AI-generated colors — bright green, red, purple, and cyan rings. I rejected it for two reasons: those clashing colors compete with my code for attention (the exact "AI-slop" the brief warns about), and they don't match my calm burgundy/rose identity kit, so the page would stop feeling like one intentional site. I redrew the same structure in my own palette instead. The information is identical; the difference is that the calm version frames my work rather than shouting over it.

**Where I chose a real capture over AI:** for the code itself, I used a real IDE screenshot rather than any generated or stylized "code-looking" image. A recruiter should see my actual work, and a real capture is always more credible than a generated stand-in.
