# FL-03: Decide Once — Build Your Identity Kit

**Track:** General AI Fluency | **Week:** 3 | **Phase:** Foundations
**Intern:** Suana Mešić — Junior Backend Developer

---

## The idea

The design is the frame, not the painting. My work is the painting. This identity kit is a small set of decisions made once and repeated, so the portfolio feels intentional and my code stays the loudest thing on the page. As a backend developer (not a designer), I kept it deliberately calm.

**Mood:** warm and rich, not cold — approachable and human.

---

## Type

One characterful pairing, both free on Google Fonts:

- **Fraunces** (headings) — a warm, characterful serif that reads human without looking unprofessional.
- **Inter** (body) — clean, readable, modern; ideal for text and code.

---

## Palette

Four colors, all contrast-checked against the background on the WebAIM contrast checker (target 4.5:1 for normal text):

| Role | Hex | Contrast on background | Result |
|------|-----|------------------------|--------|
| Text | `#2A2024` | very high (near-black) | pass |
| Main | `#6E2A46` | 9.52:1 | pass |
| Accent (links) | `#B04A67` | 4.93:1 | pass |
| Background | `#FDF7F8` | — | — |

The accent started as `#C25E7A` but only reached 3.83:1, so I darkened it to `#B04A67` to pass 4.5:1 — a deliberate readability fix, not a guess.

## Logo / Favicon

- **Wordmark:** "Suana." set in Fraunces, with the period in the main burgundy.

## Style note

> Fonts: Fraunces (headings) + Inter (body).
> Colors: text `#2A2024` on background `#FDF7F8`, main `#6E2A46`, accent `#B04A67` (links).
> Mood: warm and rich, not cold — a calm frame so the code stays the loudest thing on the page.

This note is short enough to paste back into my AI workspace before building any section, so everything stays consistent.
